Quem somos nós?

A Stevo.Chat surgiu para resolver nossas próprias necessidades de comunicação, e hoje oferecemos uma solução completa que vai além do básico. Com a nossa ferramenta, você pode enviar áudios, realizar ligações, gerenciar grupos e manter as fotos sempre atualizadas, garantindo uma experiência completa para quem usa o GHL. O que começou como uma ferramenta interna, hoje é uma solução completa, pensada para otimizar a comunicação da sua empresa.

Getting Started / Introduction

Introduction

Stevo.chat is a project dedicated to empowering small businesses, entrepreneurs, freelancers and individuals with limited resources.

Our mission is to provide a WhatsApp™ messaging solution via API, allowing these groups to strengthen their local or online businesses.

The best part is that our service is completely free, designed to support those who strive to succeed in a competitive market scenario.

Access our repository and join our community to be part of the project.

Quick Start

ℹ️

You will need to have Docker installed on your machine, see the Official Docker Documentation

To run the test version and test the main API features, copy the command below and run it in the terminal: 

docker run --name stevo-chat \
  -p 8080:8080 \
  -e WEBHOOK_URL=https://yoursite.com/webhook \
  -e WEBHOOK_BY_EVENTS=false \
  -e WEBHOOK_BASE64=false \
  -e CONFIG_SESSION_PHONE_CLIENT=Stevo.chat \
  -e CONFIG_SESSION_PHONE_NAME=Chrome \
  -e QRCODE_LIMIT=30 \
  -e AUTHENTICATION_TYPE=apikey \
  -e AUTHENTICATION_API_KEY=your-api-key-here \
  -e AUTHENTICATION_EXPOSE_IN_FETCH_INSTANCES=true \
  -e LANGUAGE=en-US \
  stevo-chat/api:latest

This command will start Stevo.chat on port 8080. Access http://localhost:8080 to see the API documentation.
