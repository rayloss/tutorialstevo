Criar Primeira Instância

Gera e retorna o QR code para conexão com WhatsApp na sua primeira instância Stevo.chat

POST /stevo-chat/instance/create/{instanceName} 
curl --request POST \
  --url https://{stevo-chat-server}/stevo-chat/instance/create/{instanceName} \
  --header 'Authorization: Bearer <stevo-chat-api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "instanceName": "minha-primeira-instancia",
    "webhook": "https://meusite.com/webhook",
    "webhookByEvents": false,
    "webhookBase64": false,
    "events": [
      "APPLICATION_STARTUP",
      "QRCODE_UPDATED",
      "MESSAGES_UPSERT",
      "MESSAGES_UPDATE",
      "MESSAGES_DELETE",
      "SEND_MESSAGE",
      "CONTACTS_SET",
      "CONTACTS_UPSERT",
      "CONTACTS_UPDATE",
      "PRESENCE_UPDATE",
      "CHATS_SET",
      "CHATS_UPSERT",
      "CHATS_UPDATE",
      "CHATS_DELETE",
      "GROUPS_UPSERT",
      "GROUP_UPDATE",
      "GROUP_PARTICIPANTS_UPDATE",
      "CONNECTION_UPDATE",
      "LABELS_EDIT",
      "LABELS_ASSOCIATION",
      "CALL_RECEIVED"
    ]
  }'

Autorização

Authorization string required
Sua chave de autorização Bearer Token do Stevo.chat

Parâmetros do Path

instanceName string required
Nome da instância a ser criada (ex: minha-primeira-instancia)

Parâmetros do Body

webhook string
URL do webhook para receber eventos da instância
webhookByEvents boolean
Se true, envia eventos específicos para diferentes endpoints
events array
Lista de eventos para escutar na instância
